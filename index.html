How CarbonIQ crawls, sanitizes, and streams real-time compliance news to your dashboard terminal.

System Topology
Carbon Intelligence Pipeline
🔍
1. Keyword Ingestion
Queries global news indexes using GNews API v4. Targets compliance keywords: "carbon credits", "ETS", "carbon trading", "CBAM", and "cap and trade".
🛡️
2. Vercel Serverless Proxy
Routes requests through a secure Node.js backend middleware (/api/news.js). Protects our private API key, injects CORS credentials, and caches queries.
⚡
3. Client-Side Fallback
Features a double-mode loading sequence. If the serverless route is blocked or offline, client-side scripts seamlessly shift to query the GNews endpoint directly.

export default async function handler(req, res) {
  // Set CORS headers for remote compliance
  res.setHeader('Access-Control-Allow-Origin', '*');
  res.setHeader('Access-Control-Allow-Methods', 'GET,OPTIONS');
  
  // GNews Query setup
  const apiKey = '8203ca35fa43...';
  const query = '("carbon credits" OR "carbon permits" OR "ETS")';  
  const url = `https://gnews.io/api/v4/search?q=${encodeURIComponent(query)}&apikey=${apiKey}`;

  try {
    const response = await fetch(url);
    const data = await response.json();
    res.status(200).json(data);
  } catch (error) {
    res.status(500).json({ errors: ['Failed to fetch news'] });
  }
}
window.fetchCarbonNews = async function() {
  const url = '/api/news';
  try {
    let response = await fetch(url);
    let data;
    if (response.ok) {
      data = await response.json();
    } else {
      // Local Static Server (Live Server) Fallback
      const directUrl = `https://gnews.io/api/v4/search?q=${query}&apikey=${apiKey}`;
      response = fetch(directUrl);
      data = await response.json();
    }
    renderNewsToDOM(data.articles);
  } catch (err) {
    console.error("News fetch failed", err);
  }
}
